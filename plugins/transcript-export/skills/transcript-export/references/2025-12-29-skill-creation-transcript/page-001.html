# Claude Code Transcripts Tool Overview

## What It Does

This tool converts Claude Code session files into clean, mobile-friendly HTML pages with pagination. It transforms JSON or JSONL session data into browseable multi-page transcripts suitable for sharing and archiving.

## Installation

Install using the `uv` package manager:

```bash
uv tool install claude-code-transcripts
```

Alternatively, run without installing:

```bash
uvx claude-code-transcripts --help
```

## Command Line Usage

The tool offers four primary commands:

- **local** (default): Select from sessions in `~/.claude/projects`
- **web**: Import sessions via Claude API
- **json**: Convert specific JSON/JSONL files
- **all**: Convert entire local session archive to browsable index

### Basic Usage

```bash
claude-code-transcripts              # Interactive picker, opens in browser
claude-code-transcripts local --limit 20
claude-code-transcripts web SESSION_ID
claude-code-transcripts json session.json -o output-directory/
```

## Output Options

All commands support:

- `-o, --output DIRECTORY`: Specify output location
- `-a, --output-auto`: Auto-name subdirectory by session ID
- `--open`: Launch in default browser
- `--gist`: Upload to GitHub Gist with preview URL
- `--json`: Include original session file
- `--repo OWNER/NAME`: GitHub repository for commit links

Generated output includes:
- `index.html`: Timeline of prompts and commits
- `page-001.html`, `page-002.html`, etc.: Paginated transcript pages

## GitHub Gist Publishing

The `--gist` flag uploads files to a GitHub Gist and provides a shareable preview URL via gistpreview.github.io. Requires GitHub CLI installed and authenticated (`gh auth login`).

Example output:
```
Gist: https://gist.github.com/username/abc123def456
Preview: https://gistpreview.github.io/?abc123def456/index.html
```

Combine with `-o` to maintain local copies alongside cloud publishing.

